# Simple HTTP Server — Project Description

A TypeScript HTTP server project for the [Boot.dev "Learn HTTP Servers in TypeScript"](https://www.boot.dev/courses/learn-http-servers-typescript) course.

---

## Goal

Build web servers in TypeScript with Express.js. By the end, you’ll be able to:

- Serve static files
- Build RESTful JSON APIs
- Add authentication and authorization

---

## Tech Stack

| Layer     | Tech       | When       |
| --------- | ---------- | ---------- |
| Runtime   | Node.js    | Throughout |
| Language  | TypeScript | Throughout |
| Framework | Express.js | Throughout |
| Database  | PostgreSQL | Chapter 6+ |
| ORM       | Drizzle    | Chapter 6+ |
| Auth      | JWT        | Chapter 7+ |

---

## Course Chapters (10)

1. **Servers** – Web server basics, why TypeScript fits
2. **Routing** – HTTP routing, Express routing
3. **Architecture** – Web architectures and how to pick one
4. **JSON** – Parsing and sending JSON
5. **Error Handling** – Handling errors in a TypeScript server
6. **Storage** – PostgreSQL, migrations, data storage
7. **Authentication** – JWT-based auth
8. **Authorization** – Implementing authorization
9. **Webhooks** – HTTP webhooks in apps
10. **Documentation** – API docs and tooling
